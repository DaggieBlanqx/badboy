# Badboy
Crawl apps from Google Play &amp; iTunes

# How to use

## Google Play

```javascript

var Badboy = require('badboy')
var app = yield Badboy.getAndroidApp("Android app package here")

```

## iTunes

```javascript

var Badboy = require('badboy')
var app = yield Badboy.getiOSApp("iTunes app id here")

```
